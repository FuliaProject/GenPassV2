# GenPassV2
Generator Password With Another Language
# Download File
```
pkg install python
pkg install github
git clone github.com/Vanilla128/GenPassV2
cd GenPassV2/GenPassLang
```
# Run File
English:
```
python3 GenPassEng.py
```
Indonesia:
```
python3 GenPassID.py
```
Russia:
```
python3 GenPassR.py
```
China:
```
python3 GenPassC.py
```
